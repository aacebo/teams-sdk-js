# Teams SDK: Javascript

a suite of packages used to build on the Teams Platform.

## Scripts

> all scripts can be run for a particular workspace by appending `--workspace=${name}`

### Build

```bash
$: npm run build
```

### Clean

```bash
$: npm run clean
```

### Format

```bash
$: npm run fmt
```

### Dev

> for samples only

```bash
$: npm run dev
```
